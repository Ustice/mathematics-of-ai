# Issue Tracker

Issues and PRDs for this repository live in GitHub Issues under `Ustice/mathematics-of-ai`. Use the `gh` CLI from this checkout so the repository is inferred from the configured remote.

## Conventions

- Create an issue with `gh issue create`.
- Read an issue and its discussion with `gh issue view <number> --comments`.
- List and filter work with `gh issue list` and structured `--json` output.
- Comment with `gh issue comment <number>`.
- Apply or remove labels with `gh issue edit <number>`.
- Close completed or rejected work with `gh issue close <number>` and a concise explanation.

When a skill says to publish a PRD or issue, create a GitHub issue in this repository.
