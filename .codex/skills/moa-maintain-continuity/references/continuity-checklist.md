# Continuity Checklist

Use this checklist after each real lesson.

## Course State

- Current phase still accurate?
- Completed lesson list updated?
- Next topic named without inventing a fake lesson artifact?
- Artifact relationships match `data/lesson-artifacts.json`?

## Notation

- New symbols introduced?
- Symbol meanings stable beyond one lesson?
- Any overloaded symbols that need local-warning notes?
- Should `reference/notation.md` or `reference/symbol-table.md` be updated?

## Glossary

- New durable terms?
- Existing definitions refined by the lesson?
- Terms that should stay lesson-local instead of global?
- Should `reference/glossary.md` or `reference/mathematical-vocabulary.md` be updated?

## Learning Journal

- New friction point?
- New teaching adjustment?
- Concept that now has strong intuition?
- Concept that needs reinforcement?
- Should this be a new `learning-journal/` entry, or a course-level update to preserved learning notes?
