# Exercise Patterns

Use a balanced ladder for most lessons.

## Ladder

1. Meaning check
   - Ask for a plain-language explanation.
   - Ask what a symbol or object represents.

2. Diagram or geometry prompt
   - Ask for a sketch when spatial intuition matters.
   - Ask what changes when a parameter changes.

3. Mechanical fluency
   - Use small numbers.
   - Show dimensions.
   - Keep the computation short enough to fit on one Notability page.

4. Derivation
   - Ask for the important step, not every algebraic expansion.
   - Name intermediate quantities to reduce working-memory load.

5. ML connection
   - Tie the math object to fitting, generalization, uncertainty, optimization, or representation.

6. Reflection
   - Ask what felt confusing.
   - Ask which vocabulary needs reinforcement.

## Medium Choice

- Use Notability for derivations, sketches, proofs, and multi-step explanations.
- Use HTML for prompts, hints, solutions, indexes, and small self-checks.
- Use widgets for concepts where changing a parameter teaches the idea faster than static text.
