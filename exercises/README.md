# Exercises

This directory contains exported handwritten exercise images organized by lesson.

## Artifact Convention

Store each lesson under `lesson-NNN-slug/`, with content pages named `exercise-01.jpg`, `exercise-02.jpg`, and so on in reading order. Optional `review.md` and implementation files belong in the same module. Omit blank export pages.

Each lesson directory is registered once as an Exercise Artifact Module in [lesson-artifacts.json](../data/lesson-artifacts.json); image order is derived from numeric filenames.

## Review Workflow

After Jason submits exercise images:

- Review the mathematical work.
- Explain corrections or useful refinements.
- Decide whether the work demonstrates readiness for the next lesson.
- Add the nonblank images to the lesson's Exercise Artifact Module.
- Continue or create the next lesson when the review passes.

## Format Rule

Do not create new exercise Markdown files. Existing Markdown files are retained as historical review notes only.
