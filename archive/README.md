# Archive

This directory preserves superseded course and reconstruction material that no active application or lesson-production workflow consumes.

## Contents

- `reference/` contains the pre-canonical notation and glossary snapshots.
- `learning-journal/` contains the two journals superseded by lesson-specific entries under `learning-journal/` and the current learning model under `reference/`.
- `lesson-source-index/` contains the static index superseded by the Lesson Catalog and Astro routes.
- `reconstruction-notes/` contains coordination notes from the completed Lessons 1-25 archive backfill.

Do not update archived files as current authorities. Preserve them for historical context only.
