# Curriculum

This is the working roadmap for learning the mathematics needed to read modern machine learning and AI research.

## Phase I — Linear Algebra Foundations

1. Vectors
2. Matrices as Transformations
3. Matrix Multiplication Concepts
4. Determinants
5. Inverse Matrices
6. Eigenvectors and Eigenvalues
7. Change of Basis
8. Projection
9. Least Squares
10. Singular Value Decomposition
11. Principal Component Analysis

## Phase II — Probability, Statistics, and Geometry

12. Covariance
13. Probability as Linear Algebra
14. Covariance Geometry
15. Multivariate Gaussian
16. Maximum Likelihood Estimation
17. Loss Functions
18. MAP Estimation
19. Bayesian Estimation Introduction
20. Bayesian Estimation Update
21. Bayesian Predictive Distributions

## Phase III — Generalization and Optimization

22. Model Complexity
23. Regularization
24. Early Stopping
25. Dropout
26. Gradient Descent
27. Stochastic Gradient Descent
28. Momentum
29. Adaptive Optimizers
30. Learning Rate Schedules

## Later Phases

Planned later phases include neural networks, backpropagation, information theory, attention and transformers, optimization pathologies, probabilistic modeling, and guided paper-reading.

## Public Repo Policy

This repository should contain original course material, exercises, notation, diagrams, and public-safe learning reflections. Avoid committing private personal details, credentials, copyrighted textbook excerpts, or raw source material that is not ours to publish.
