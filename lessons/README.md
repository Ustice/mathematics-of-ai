# Lessons

This directory is the workspace for lesson material.

## Polished Lessons

Future polished lessons should use [lesson-template.md](lesson-template.md). Each lesson that introduces formulas should include a **Notation / Symbol Table** section so the learner does not have to reconstruct symbol meanings from memory.

Polished lessons should include:

- Goal.
- Motivation.
- Prerequisites.
- Notation / symbol table.
- Core ideas.
- Worked examples.
- Why the idea shows up in ML.
- Exercises.
- Implementation exercise.
- Reflection questions.

## Raw Source Material

Exported ChatGPT lesson transcripts currently live in [transcripts](transcripts/). Treat them as source material, not final lessons.

The public lesson index is [index.html](index.html). The machine-readable source map is [../data/lesson-sources.json](../data/lesson-sources.json).

## Status Rule

Do not mark a lesson complete merely because a transcript exists. A completed polished lesson should be edited, internally coherent, and linked to exercises, implementation notes, and reference entries where relevant.
