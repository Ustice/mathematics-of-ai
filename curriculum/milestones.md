# Milestones

Milestones describe evidence of durable understanding. They should be marked complete only when the repo contains enough lesson, exercise, implementation, or paper-reading material to support the claim.

## M1 — PCA and Logistic Regression

Evidence of progress:

- Can explain projections, covariance, eigenvectors, and SVD geometrically.
- Can implement PCA from first principles.
- Can implement logistic regression from first principles.
- Can explain what optimization problem each algorithm solves.

## M2 — Small Neural Network From Scratch

Evidence of progress:

- Can explain a neural network as a composition of parameterized functions.
- Can implement dense layers, activation functions, forward passes, losses, and training loops.
- Can explain how gradients move through a computational graph.
- Can diagnose underfitting, overfitting, and simple optimization failures.

## M3 — Reverse-Mode Automatic Differentiation

Evidence of progress:

- Can explain the difference between symbolic differentiation, numerical differentiation, forward-mode autodiff, and reverse-mode autodiff.
- Can represent computations as a graph of primitive operations.
- Can implement reverse accumulation of gradients.
- Can validate gradients against finite differences on small examples.

## M4 — Multi-Head Attention

Evidence of progress:

- Can explain queries, keys, values, attention scores, softmax, heads, and output projections.
- Can implement scaled dot-product attention from first principles.
- Can implement multi-head attention with explicit tensor shapes.
- Can connect attention to weighted averaging, similarity, and information routing.

## M5 — Classic Transformer Paper Reproduction

Evidence of progress:

- Can read a classic transformer paper and identify its mathematical assumptions.
- Can reproduce the core attention argument and architecture diagram in personal words.
- Can implement a minimal transformer block.
- Can compare implementation behavior against the paper's claims at small scale.

## M6 — Independent Recent Paper Evaluation

Evidence of progress:

- Can read a recent ML paper without relying on a tutorial walkthrough.
- Can identify the paper's assumptions, main mathematical tools, and central claim.
- Can distinguish experimental evidence from mathematical argument.
- Can implement or sketch the core idea well enough to evaluate its plausibility.
- Can write an independent critique with open questions and limitations.
