# Implementation

This directory tracks implementation work that supports mathematical understanding.

## Companion Project

The companion implementation project is described in [project-plan.md](project-plan.md).

Implementation is a learning tool here, not a production-quality numerical library. Prefer clear first-principles TypeScript / JavaScript code over speed or broad framework coverage at the beginning.

## Working Principles

- Use TypeScript / JavaScript first.
- Avoid ML frameworks until the underlying math is visible.
- Write code that exposes shapes, symbols, assumptions, and objective functions.
- Add comparisons to PyTorch, JAX, or other tools later only when they clarify behavior.
