# Learning Journal

This directory records durable learning observations for the self-learning program.

Use the journal to record:

- Lesson summaries.
- What clicked.
- What was confusing.
- Notation issues.
- Concepts to reinforce later.
- Connections made across topics.
- Changes to the curriculum.

## Template

Use [journal-template.md](journal-template.md) for new entries.

Existing notes are preserved in [../learning-journal.md](../learning-journal.md) and [../journal/learning-journal.md](../journal/learning-journal.md).
